
SqlQueryCreatorJpa


Sql_Jpa_Hibernate application для получения данных из таблицы БД и создания файлов с данными в формате .txt, .sql.


♣ ФУНКЦИОНАЛ ♣

1. Приложение подключается к таблице из БД
2. Выборка данных из таблицы БД
3. Формирование файла выгрузки в формате .txt или .sql
4. Из столбцов id и title формируется название сохраняемого файла
5. Из столбца request формируется содержимое файла
6. Файл сохраняется в указанной папке по указанному пути (например на рабочий стол)
7. Каждая строка в таблице БД формирует отдельный файл для выгрузки.

8. Интеграция с базой данных - Использование PostgreSQL
9. Реализована возможность вносить в таблицу БД новые данные
10. Реализована возможность получать/читать данные из таблицы БД по id
11. Реализована возможность получать/читать ВСЕ данные из таблицы БД (СПИСКОМ)
12. Реализована возможность изменять данные в таблице БД
13. Реализована возможность удалять данные из таблицы в БД
